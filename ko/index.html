Build fast. Run Fast.

비동기 Python 3.7+ 웹 서버/프레임워크

시작하기 →

간단하고 가볍게

스마트한 기본값과 부풀림이 없는 직관적인 API를 통해 앱을 구축하기 위해 바로 작업을 시작할 수 있습니다.

제약받지 않고 유연하게

툴링에 제약을 받지 않고 원하는 방식으로 설계할 수 있습니다.

성능 및 확장성

초기부터 속도와 확장성을 주 삼아 설계되었습니다. 크고 작은 웹 애플리케이션을 지원할 준비가 되어있습니다.

프로덕션 준비

웹 애플리케이션을 구동할 준비가 된 웹 서버가 기본으로 제공됩니다.

수백만 명의 신뢰

Sanic은 PyPI에서 전반적으로 가장 인기있는 프레임 워크 중 하나이며 비동기를 지원하는 최고의 프레임워크입니다.

커뮤니티 중심

이 프로젝트는 커뮤니티를 위해 커뮤니티에서 유지되고 운영됩니다.
